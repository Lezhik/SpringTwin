## Описание

Создать базовый layout приложения и систему навигации между модулями. Layout должен включать:

1. Header с логотипом и навигацией
2. Sidebar с меню модулей
3. Main область для контента
4. Footer с информацией о приложении

### Компоненты для создания

- `AppLayout.vue` - Основной layout приложения
- `AppHeader.vue` - Header компонент
- `AppSidebar.vue` - Sidebar с навигацией
- `AppFooter.vue` - Footer компонент
- `NavItem.vue` - Элемент навигации

### Маршруты

- `/` - Главная страница (dashboard)
- `/project` - Модуль project
- `/architecture` - Модуль architecture
- `/analysis` - Модуль analysis
- `/report` - Модуль report
- `/mcp` - Модуль mcp

## Примечания

- Использовать Vue Router 4.x для Vue 3
- Использовать lazy loading для маршрутов: `() => import('./views/ModuleView.vue')`
- Использовать семантические теги: header, nav, main, footer, aside
- Использовать атрибуты name и role для семантической разметки
- Добавить aria-labels для доступности
